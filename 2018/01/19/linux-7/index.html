linux——中文乱码处理

SecureCRT连接远程服务器后，发现有中文乱码问题，下面是解决方方案。

  1. 查看文件编码,打开文件，输入

    :set fileencoding

    结果显示fileencoding=cp936

  2. 修改文件编码，输入

    :set fileencoding=utf-8

    再次输入1中的语句检查文件编码，显示fileencoding=utf-8

  3. 编辑vimrc，在终端输入

    vim ~/.vimrc

    在最后一行添加

    set encoding=utf-8 fileencodings=ucs-bom,utf-8,cp936

至此结束。
