Reading - Enriching BERT with Knowledge Graph for Document Classification


Picture downloaded from here.

Reference

  • Paper: Enriching BERT with Knowledge Graph for Document Classification
  • Github: https://github.com/malteos/pytorch-bert-document-classification

Hightlights

Features in model

Three features:

  • BERT embedding
    Raw text embedding using pre-trained BERT.
    Parameters: Dimension=768.
  • Meta Data
    Some statistical features
    Parameters: dimension=20
  • Knowledge Embedding
    Entities embedding using PyTorch BigGraph.
    Parameters: dimension=200.

Labels

Two sub-tasks in model:

  • Sub-task A: multi-label with 8 top-level classes
  • Sub-task B: multi-label with 234 detailed classes

Architecture

Briefly, resembles the original BERT model, and combines text- and non-text features with a multilayer perceptron.
