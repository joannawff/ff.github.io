Overview

Better Data!=More Data

Better data is always better. Improve data quality is alwayss well invested.

Tip: doing some form of smart sampling on your population the right way (e.g. using stratified sampling) can get you to better results than if you used the whole unfiltered data set.

Data Without a Sound Approach = Noise

The availability of data enables more and better insights and applications. More data indeed enables and requires better approaches

Attributes related to data quality

  • Accuracy
  • Consistency
  • Timeless
  • Completeness

Ways to improve data quality

  • Resolve missing values
  • Convert the Date feature column to DateTime format
  • Parse date/time features
  • Remove unwanted values(characters)
  • Convert categorical columns to “one-hot encodings”
